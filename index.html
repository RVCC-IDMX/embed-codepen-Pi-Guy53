Pen: Password Complexity Field

by: TwiggyBuffalo

This pen showcases a system to chech for the complexity of a password. The HTML and CSS are very simple, but its the JavaScript that makes this work.

Bascially what the code is doing, is checking for a series of parameters in the entered password. Each parameter has a score associated with it, and the total score is used to display the password strength
